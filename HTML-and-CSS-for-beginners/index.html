Introduction to HTML

(HYPERTEXT MARK-UP LANGUAGE)

  • HTML is a mark-up language, providing structure to the document
  • It is written in the form of HTML elements
  • Each element is composed of tags enclosed in angle brackets, and everything inbetween
This is a paragraph element
INTRODUCTION TO HTML

Document type element (doctype)

  • Is very first element in an HTML document
  • It let the web browser know about what version of the markup language the page is written in
  • In HTML5 we use the following doctype declaration:
<!DOCTYPE html>
INTRODUCTION TO HTML

Document structure

All HTML pages are composed of two main parts:

  • <head>
  • <body>
INTRODUCTION TO HTML

INTRODUCTION TO HTML

<body>

Introduction to CSS

(Cascading Stylesheets)

  • CSS is independent from HTML
  • CSS is responsible for the look and feel of a document
  • Enables you to change the presentation of elements, such as colors, layout and fonts
  • Allows you to style for different screen sizes and devices
INTRODUCTION TO CSS

Basic syntax

The CSS declaration has the following format:

selector {
  property: value;
}
  • Selector: HTML tag to which the property apply
  • Property: The property that will be defined
  • Value: The value of the property
INTRODUCTION TO CSS

Basic syntax example

file: style.css 
body {
  background-color: lightblue;
}
  • body = selector
  • background-color = property
  • lightblue = value
INTRODUCTION TO CSS

Styling your elements

file: style.css 
h1 {
  background-color: red;
  color: grey;
  margin-top: 40px;
  text-align: center;
}

Images

There are two main types of images:

  • Furniture images: are the images necessary for the rendering of layout or component
  • Content images: are the images used to complement and cotextualise your content
IMAGES

Furniture images

  • Are content-agnostic
  • Can be shared amongst components
  • Normally are declared in the stylesheet for greater reusability
  • Examples: background images, navigation arrows, close icons, etc
IMAGES

Adding furniture images

file: style.css 
body {
  font-family: "Open Sans", sans-serif;

  background-image: url('https://tinyurl.com/jnm5v38');
  background-position: top left;
  background-repeat: repeat;
  color: white;
}
Image credit © Iwan Gabovitch
IMAGES

Content images

  • Content images are added within the context of the page (HTML file)
  • <IMG> element source (src): image path, which can be local or external
  • <ALT> attribute: is shown instead of the image, if the image cannot be displayed. The alt attribute also describes the image for screen readers
  • <TITLE> attribute: is shown alongside the image, as a tooltip
